Chrome 확장 프로그램

검색 결과를 클릭하기 전에,
AI가 먼저 확인합니다

구글 검색 결과에서 SEO 포이즈닝, 피싱 사이트, 경쟁사 우회 광고를 실시간으로 탐지합니다. 의심스러운 링크는 블러 처리해 실수 클릭을 방지합니다.

Chrome에 설치 GitHub 소스 보기
신뢰도 등급

5단계로 한눈에 확인

AI가 도메인과 검색 의도를 분석해 각 결과에 등급을 부여합니다.

공식 배포처

OFFICIAL

검색 키워드의 공식 도메인 또는 공식 앱스토어

신뢰할 수 있음

TRUSTED

나무위키, GitHub 등 공인된 신뢰 플랫폼

알 수 없음

UNKNOWN

블로그·커뮤니티 등 신뢰도 불명확

주의

WARNING

비공식 배포처, 경쟁사 우회 광고

위험

DANGER

공식 브랜드 사칭, SEO 포이즈닝 의심 도메인

스크린샷

실제 동작 화면

카카오톡, 클로드 등 실제 검색에서 탐지된 결과입니다.

사용 방법

3단계면 충분합니다

Chrome 웹스토어에서 바로 설치할 수 있습니다.

1

확장 프로그램 설치

Chrome 웹스토어에서 클릭전을 설치합니다.

2

Google 검색 후 분석 요청

원하는 키워드를 검색한 뒤, 툴바의 클릭전 아이콘을 클릭하고 "지금 분석하기"를 누릅니다.

3

등급 확인 후 안전하게 클릭

AI 분석이 완료되면 각 결과에 등급 배지가 표시됩니다. 블러 처리된 결과는 클릭해서 내용을 확인할 수 있습니다.
